    Hands-On Demo #

    Overview #

    This closing section is a hands-on, live demonstration. Attendees follow along with provided Jupyter notebooks to generate kernels with LLM agents: first Triton kernels for NVIDIA GPUs, then NKI kernels for AWS Trainium. We finish with an interactive end-to-end walkthrough and open Q&A, tying together the post-training, reasoning, and agentic methods from the earlier sections.
